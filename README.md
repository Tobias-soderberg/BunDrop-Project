Tror min git ligger lite fel (utanför projektet) så behöver då in en mapp till för att kunna köra commands.
1. På valfri plats (cmd eller terminalen i VS Code) kör kommando "npx json-server --watch db.json --port 3001" då jag använder port 3001.
2. I en annan terminal (cmd), navigera till projektet och kör "npm i" om inte installerat, sedan "npm dev run" för att köra appen.
